# 🎱 Billiard Hall Management & POS System

A desktop management software designed to automate daily operations, table availability tracking, and billing for a commercial billiard hall.

## 🚀 Key Features
* **Table Management:** Real-time tracking and time/rate calculation for 6 billiard tables (60,000 UZS/hour).
* **Automated Billing:** Instant bill computation based on exact playing time.
* **Bar Inventory Control:** Admin panel to manage bar sales and integrate product expenses directly into the customer's final invoice.
* **Security:** Login authentication for system administrators.

## 🛠 Tech Stack
* **Language:** Python
* **GUI / Architecture:** Object-Oriented Programming (OOP)
* **Database / Storage:** Local File/JSON Database integration
